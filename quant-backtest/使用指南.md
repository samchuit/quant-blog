# 量化交易系统使用指南

## 已完成的功能

### 1. 币安模拟盘交易 ✅
**文件**: `live_trading.py`

**功能**:
- 接入币安Testnet模拟交易
- 自动获取实时数据
- 模型预测交易动作
- 支持多个币种同时交易

**使用方法**:
```bash
# 模拟运行（无需API Key）
python live_trading.py

# 真实模拟盘（需配置API Key）
# 编辑文件首部的 API_KEY 和 API_SECRET
```

**注意事项**:
- 需要去币安测试网申请API Key: https://testnet.binance.vision/
- 测试网提供虚拟资金，无真实风险

---

### 2. 自动月度重训练 ✅
**文件**: `auto_retrain.py`

**功能**:
- 每月1号自动重新训练模型
- 使用最新数据保持模型更新
- 记录训练日志

**使用方法**:
```bash
# 立即执行一次训练
python auto_retrain.py --mode now

# 启动定时任务（每月自动重训练）
python auto_retrain.py --mode schedule
```

**建议**:
- 每月初检查训练结果
- 如果效果下降，可增加训练步数
- 保持模型与市场同步

---

### 3. 监控报告系统 ✅
**文件**: `monitor_report.py`

**功能**:
- 每日生成策略表现报告
- 显示价格、技术指标
- 提供模型交易建议

**使用方法**:
```bash
# 生成一次性报告
python monitor_report.py

# 启动每日自动报告
python monitor_report.py --mode daily
```

**报告内容**:
- 各币种当前价格
- 日/30日涨跌幅
- RSI、布林带等技术指标
- 模型建议交易动作

## 配置说明

### API配置
编辑 `live_trading.py`:
```python
API_KEY = "你的币安测试网API Key"
API_SECRET = "你的币安测试网Secret"
```

### 定时配置
编辑 `auto_retrain.py`:
```python
# 修改训练时间（默认每月1号凌晨3点）
schedule.every().day.at("03:00").do(check_and_retrain)
```

### 报告频率
编辑 `monitor_report.py`:
```python
# 每日报告时间
schedule.every().day.at("09:00").do(generate_report)
schedule.every().day.at("21:00").do(generate_report)
```

## 推荐工作流程

### 每日
1. 查看监控报告
2. 检查模型建议
3. 如有异常，手动排查

### 每周
1. 回顾本周表现
2. 对比策略收益 vs BH收益
3. 记录异常情况

### 每月
1. 执行模型重训练
2. 验证新模型效果
3. 更新备份

---

## 注意事项

1. **API安全**: 不要泄露API Key和Secret
2. **风险控制**: 先用模拟盘验证，再用实盘
3. **数据备份**: 定期备份模型和日志
4. **市场监控**: 极端行情时关注策略表现
