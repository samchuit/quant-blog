# 量化交易系统 - 完整使用指南

## 📦 系统功能

### ✅ 已实现功能

1. **自动交易** - 模型预测 + 自动下单
2. **日志记录** - 所有交易记录保存
3. **监控报告** - 每日生成报告
4. **定时运行** - 可设置定时任务

---

## 🚀 快速启动

### 方法一：启动面板
```bash
双击运行: start_monitor.bat
```

### 方法二：命令行
```bash
# 运行一次
python trading_monitor.py --mode once

# 定时运行 (每60分钟)
python trading_monitor.py --mode schedule --interval 60

# 定时运行 (每30分钟)
python trading_monitor.py --mode schedule --interval 30
```

## ⚙️ 配置说明

### API配置
在 `trading_monitor.py` 中修改:
```python
API_KEY = "你的API Key"
API_SECRET = "你的API Secret"
```

### 交易币种
在 `trading_monitor.py` 中修改:
```python
symbols = ['BTCUSDT', 'ETHUSDT', 'SOLUSDT']  # 可添加更多
```

### 运行间隔
```bash
python trading_monitor.py --mode schedule --interval 30  # 30分钟
python trading_monitor.py --mode schedule --interval 60  # 60分钟
```

## 🔧 故障排除

### 问题：下单失败
- 检查API Key是否正确
- 检查账户余额是否充足
- 检查网络连接

### 问题：模型预测异常
- 确认模型文件存在
- 检查数据获取是否正常
